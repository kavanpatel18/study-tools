# 🧠 Study Tools

A collection of beautifully designed, browser-based study tools — no frameworks, no installs, just open and use.

## Pages

| Tool | Description |
|------|-------------|
| [Memory Mastery Guide](memory-techniques-guide.html) | Step-by-step interactive guide to 6 research-backed memory techniques for exam prep |
| [Exam Countdown](exam-countdown.html) | Mission-control style countdown tracker — add exams, watch the clock tick |
| [Flashcard Maker](flashcard-maker.html) | Build decks, flip cards, track your score in study mode |

## Features

- **No dependencies** — pure HTML, CSS, and vanilla JS
- **Persistent storage** — flashcards and exam dates save to localStorage
- **Fully responsive** — works on desktop and mobile
- **Distinct designs** — each tool has its own unique visual identity

## Live Demo

Hosted via GitHub Pages: `https://kavanpatel18.github.io/study-tools/`

## Getting Started

Clone the repo and open any `.html` file directly in your browser — or visit the GitHub Pages link above.

```bash
git clone https://github.com/kavanpatel18/study-tools.git
cd study-tools
open memory-techniques-guide.html
```

## Tools Used

Built with vanilla HTML/CSS/JS and Google Fonts. No build step required.
